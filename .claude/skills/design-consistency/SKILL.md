# Design Consistency

The mockup images in `docs/Design/` contain documented inconsistencies across shape, color, navigation, and layout patterns.

**Before using any design mockup, read `docs/design_inconsistencies.md` in full.** Each entry lists the observed inconsistency and the agreed Expected Result. Always implement the Expected Result — never copy the mockup as-is.
