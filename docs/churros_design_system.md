# Churros — Design System

> **Design principles:** Warm. Calm. Friendly. Home.  
> Inspired by sticky notes, Scandinavian interiors, and the feeling of a well-organized home.

---

# Brand Personality

Churros should never feel like a productivity app.

Instead it should feel like:

- 🏡 Home
- ☀️ Warm morning light
- 📝 Sticky notes on a fridge
- 🤝 Family collaboration
- 😊 Positive reinforcement

The interface should encourage users instead of pressuring them.

---

# Design Principles

## Warm

Avoid cold grays and harsh blacks.

Use warm neutrals with subtle yellow and brown undertones.

---

## Spacious

Every screen should breathe.

Generous whitespace.

Large touch targets.

Clear hierarchy.

---

## Soft

Rounded corners.

Rounded icons.

Soft shadows.

Organic spacing.

No sharp edges.

---

## Minimal

Focus on content.

Remove unnecessary decorations.

Every element should have a purpose.

---

## Human

Micro interactions should feel rewarding.

Animations should be subtle.

Errors should be helpful.

Success should feel satisfying.

# Typography

## Primary Font

### **Manrope**

Reasons:

- Free (Google Fonts)
- Commercial use (OFL)
- Variable font
- Excellent readability
- Full Latin + Cyrillic support, including Ukrainian characters

---

## Font Scale

| Style | Weight | Size |
|--------|--------|------|
| Display Large | 700 | 40 |
| Display | 700 | 34 |
| Headline | 700 | 28 |
| Title Large | 600 | 22 |
| Title | 600 | 18 |
| Body Large | 500 | 16 |
| Body | 400 | 14 |
| Label | 600 | 13 |
| Caption | 500 | 12 |

Use a line-height of approximately **1.4×** for body text and **1.2×** for headings.

---

# Font Weights

| Weight | Usage |
|----------|------|
| 400 | Body |
| 500 | Buttons |
| 600 | Titles |
| 700 | Headlines |

Avoid using ultra-light weights.

---

# Iconography

Style:

- Rounded
- 2px stroke
- Minimal
- Simple geometry

Icons should feel approachable rather than technical.

Avoid filled icons unless representing selected state.

---

# Border Radius

| Component | Radius |
|-----------|--------|
| Buttons | 16dp |
| Cards | 20dp |
| Sticky Notes | 18dp |
| Bottom Sheets | 28dp |
| Dialogs | 24dp |
| Avatars | Circle |

---

# Elevation

Very subtle.

| Level | Usage |
|-------|------|
| 0 | Background |
| 1 | Cards |
| 2 | FAB |
| 3 | Bottom Sheet |

Prefer contrast over heavy shadows.

---

# Spacing

Use an **8dp spacing system**.

| Token | Value |
|--------|------|
| XS | 4 |
| S | 8 |
| M | 16 |
| L | 24 |
| XL | 32 |
| XXL | 48 |

Page padding:

24dp

Card padding:

16dp

Section spacing:

24dp

---

# Buttons

Primary

- Honey background
- Espresso text

Secondary

- White surface
- Espresso border

Text

- Espresso text
- No border

Danger

- Terracotta background
- White text

Buttons should always have rounded corners (16dp).

---

# Cards

Cards are the primary organizational element.

Characteristics:

- White background
- Soft shadow
- Rounded corners
- Large padding
- Minimal borders

Cards should feel like premium sticky notes rather than dashboard widgets.

---

# Motion

Animation duration:

150–250ms

Style:

- Ease Out
- Soft
- Natural

Interactions:

- Slight card lift on press
- Smooth checkbox completion
- Gentle fade transitions
- Bottom sheets slide naturally
- No flashy effects

---

# Illustrations

Style:

- Flat
- Minimal
- Friendly
- Warm colors

Avoid detailed illustrations.

Use only when enhancing empty states or onboarding.

---

# Imagery

Photography should feature:

- Families
- Cozy homes
- Natural lighting
- Everyday moments

Avoid stock-office aesthetics.

---

# Accessibility

- Minimum touch target: 48×48dp
- High color contrast
- Dynamic font sizing support
- Screen reader labels
- Color is never the only indicator of state

# Do

✅ Use whitespace generously

✅ Keep layouts simple

✅ Encourage instead of overwhelm

✅ Prioritize today's tasks

✅ Make interactions feel rewarding

---

# Don't

❌ Bright saturated colors

❌ Heavy shadows

❌ Sharp corners

❌ Busy dashboards

❌ Dense information

❌ Corporate productivity aesthetics

❌ Excessive animations

---

# Brand Motto

> **Like chores, but sweeter.**

Every interaction should reinforce this idea: helping with household tasks should feel collaborative, lightweight, and rewarding—not like work.
